# Katalon E-Commerce Automation

## Project Overview
Selenium-based web UI automation project built using Katalon Studio.
The goal of this project is to demonstrate real-world QA automation practices
including test design, positive and negative testing, and maintainable test structure.

## Application Under Test
Sauce Demo  
https://www.saucedemo.com

## Test Scope
### In Scope
- Login functionality
- Product listing
- Add to cart
- Checkout process
- Logout

### Out of Scope
- Performance testing
- Security testing
- Cross-browser testing

## Tools & Technologies
- Katalon Studio
- Selenium (via Katalon)
- Groovy
- GitHub Desktop

## Test Design Approach
- Manual test cases designed before automation
- Positive and negative scenarios covered
- Object Repository used for element management
- Assertions added for validation

## How to Run the Tests
1. Clone this repository
2. Open the project in Katalon Studio
3. Execute the test suite from the Test Suites folder
